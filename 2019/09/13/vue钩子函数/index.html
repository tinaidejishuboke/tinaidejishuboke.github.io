# 一、路由钩子函数相关概念

  • 什么情况下要使用路由的钩子函数？
    • 在路由跳转的时候，我们需要一些权限判断或者其他操作，这个时候就需要使用路由的钩子函数
  • 路由钩子函数的定义:
    • 路由钩子主要是给使用者在路由发生变化时进行一些特殊的处理而定义的函数

# 二、路由钩子的分类

# 1、全局路由钩子

在初始化 VueRouter 以后，直接使用 router 实例进行注册
包括两个钩子：beforeEach、afterEach（每个路由切换前或者切换后调用）

# 2、单个路由钩子

写在路由配置中，只有访问到这个路径，才能触发钩子函数
包括两个钩子：beforeEnter、beforeLeave

# 3、组件内路由钩子

这个钩子要在组件内定义
包括三个钩子：beforeRouteEnter、beforeRouteUpdate、beforeRouteLeave

（1） 路由对象是在使用 vue-route 启动应用时，每个匹配的组件实例都会被注入 router 的对象，称之为路由对象。
在组件内部可以通过 this.$route 的方式调用
（2）路由对象有： $route.path , $route.params, $route.query, $route.router 等属性
