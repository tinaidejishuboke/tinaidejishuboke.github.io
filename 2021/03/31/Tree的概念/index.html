# 二叉树的概念

# 概念：二叉树是树形结构的一个重要类型，二叉树特点是每个结点最多只能有两棵子树，且有左右之分，就是递归调用自己，定一个根节点，二叉树递归的最后一层，就被称为叶节点。

# 2、二叉树先序遍历就是先访问自己，然后左子树，然后右子树。

# 0,1,3,7,8,4,9,2,5,6

# 3、二叉树的中序遍历是先访问左子树，然后访问自己，最后右子树。

# 7,3,8,1,9,4,0,5,2,6
